### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/stefania-spetetchi/tennis-app
   ```
2. Install NPM packages in Frontend
   ```sh
   cd frontend
   npm install
   cd -
   ```
3. Install NPM packages in Backend
   ```sh
   cd backend
   npm install
   ```
